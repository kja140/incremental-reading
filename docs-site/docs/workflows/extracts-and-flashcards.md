# Extracts and flashcards

An **extract** is material you may want to read or refine again. A **flashcard** is a prompt you want to answer from memory. Do not make the distinction based only on length.

## Capture options

- **Extract selection** creates an extract from selected Markdown text.
- **Extract from clipboard (PDF-aware)** creates a linked extract from copied content.
- **Flashcard from clipboard** creates a Spaced Repetition card note.
- **Flashcard: name this image** creates a visual prompt.
- **Occlusion: create cards from image** creates cards that hide selected regions.
- **Export inline cards to Spaced Repetition** recognizes `Q:: … ::A::`, `{{c1::…}}`, and configured highlight forms.

## A useful rule

Extract first when context still matters. Make a card only when you can write a specific prompt with a stable answer. One strong card is better than five copied sentences.
