# PDFs and read points

The integrated PDF reader can open a PDF stored in the vault or a local external path recorded in a source note. Choose **Open PDF (Toolkit viewer)** from **Current element actions…**.

The viewer provides page navigation and saves the current page as a read point. When you run **Grade current reading topic**, the current position is retained so the next session resumes instead of restarting.

After copying text from a PDF, choose **Extract from clipboard (PDF-aware)** from **Capture or create…**. For diagrams or figures, the same menu's **Image extract from clipboard** action saves the image and creates a linked note.

:::info Desktop only
External file access and the integrated reader rely on Electron/Node APIs. EPUB is not supported by the PDF viewer.
:::
