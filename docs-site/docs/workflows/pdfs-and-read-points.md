# PDFs and read points

The integrated PDF reader can open a PDF stored in the vault or a local external path recorded in a source note. Run **Open PDF (Toolkit viewer)** from the source.

The viewer provides page navigation and saves the current page as a read point. When you grade and advance, the current position is retained so the next session resumes instead of restarting.

**Extract from clipboard (PDF-aware)** is useful after copying text from a PDF. The new extract links back to its parent source. For diagrams or figures, **Image extract from clipboard** saves the image in the configured attachments area and creates a linked note.

:::info Desktop only
External file access and the integrated reader rely on Electron/Node APIs. EPUB is not supported by the PDF viewer.
:::
