# Notes and clippings

Open **Capture or create…**. Choose **New source** when you want the Toolkit to create a fresh source note, or **Import clipping (active note)** when useful content already exists in the active note.

For long Markdown documents, **Advanced tools…** contains **Split article on H2 headings** and **Split book into chapters**. Inspect the resulting hierarchy in the Knowledge Tree and adjust parents if needed.

When reading, **Current element actions…** contains **Toggle read-point**, **Jump to read-point**, and timeline checkpoints.
