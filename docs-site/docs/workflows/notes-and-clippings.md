# Notes and clippings

Use **New source** when you want the Toolkit to create a fresh source note. Use **Import clipping (active note)** when useful content already exists in the active note.

For long Markdown documents, **Split article on H2 headings** can turn major sections into manageable child topics. **Split book into chapters** supports chapter-shaped source material. Inspect the resulting hierarchy in the Knowledge Tree and adjust parents if needed.

When reading, **Toggle read-point** marks where to resume and **Jump to read-point** returns there. Timeline checkpoints provide additional navigation landmarks without creating recall cards.
