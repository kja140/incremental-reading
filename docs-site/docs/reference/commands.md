# Command reference

| Command | Purpose |
|---|---|
| Build today's session queue | Calculate and save the priority-aware mixed queue. |
| Next element | Open the following saved queue path and automatically start SR when it is a card. |
| Grade current reading topic | Grade and reschedule the current source or extract without navigating. |
| Extract selection | Create a child extract from selected text. |
| Flashcard from clipboard | Write a native Spaced Repetition card note. |
| Capture or create… | Open source, extract, image, and occlusion creation actions. |
| Current element actions… | Open scheduling, priority, navigation, read-point, and checkpoint actions. |
| Open Toolkit view… | Open the dashboard, analytics, queue, knowledge tree, or guide. |
| Advanced tools… | Open subset, overload, split, tree, migration, export, and diagnostic tools. |
