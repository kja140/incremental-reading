# Command reference

## Capture and create

| Command | Purpose |
|---|---|
| New source | Create a scheduled source note. |
| Import clipping (active note) | Convert the active note into a source. |
| Extract selection | Create a child extract from selected text. |
| Extract from clipboard (PDF-aware) | Create a parent-linked extract from clipboard text. |
| Flashcard from clipboard | Write a native Spaced Repetition card note. |
| Flashcard: name this image | Create an image naming card. |
| Image extract from clipboard | Save an image and create a linked extract. |
| Occlusion: create cards from image | Create image-occlusion cards. |
| Export inline cards to Spaced Repetition | Convert supported inline forms to card notes. |
| Migrate legacy cards to Spaced Repetition | Convert pre-0.4 card notes once. |

## Review and scheduling

| Command | Purpose |
|---|---|
| Next element | Choose the next priority-aware learning element. |
| Random due element | Choose randomly from due topics. |
| Grade and advance | Grade the current topic, reschedule it, and continue. |
| End session (grade current) | Grade the current topic and stop. |
| Review cards | Open Spaced Repetition review. |
| Review cards in current note | Review cards associated with the active note. |
| Done | Mark complete. |
| Dismiss | Remove from active rotation. |
| Postpone | Move the due date forward. |
| Schedule (manual date) | Set an explicit due date. |
| Subset review | Review a selected portion of the collection. |
| Mercy (spread overdue) | Distribute overdue work over future dates. |
| Postpone subtree | Postpone an element and descendants. |

## Organise and navigate

Set/boost priority; open Dashboard, Reading Queue, or Knowledge Tree; create a category; move an element; open its parent; open a PDF; toggle/jump to a read point; add a timeline checkpoint; split articles or books; and show Stats.
